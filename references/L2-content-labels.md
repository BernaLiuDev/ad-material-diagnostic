# L2 内容分镜诊断：视频内容质量标签（Content Diagnostic Label）

## 诊断视角

从**内容留存漏斗**的角度审视视频素材的内容质量：

```
Impressions → 3s完播率 → 5s完播率 → 全完播率 → 停留时长
```

通过**秒级别完播数据**的形态分析，诊断视频内容在哪个时间段出了问题，以及出了什么类型的问题。

---

## 关键派生指标

在进行 L2 诊断前，计算以下指标：

| 指标 | 公式 | 含义 |
|------|------|------|
| Hook Rate | v_3s / Impressions | 前3秒抓力 |
| Hold Rate | v_5s / v_3s | 3秒后的承接力 |
| P50 留存率 | v_p50 / Impressions | 视频中点的留存水平 |
| 深度完播率 | v_p100 / v_p50 | 看完一半的人有多少看到底 |
| 3s-5s 流失率 | (v_3s - v_5s) / v_3s | 3-5秒段的用户流失比例 |

**大盘对比说明：** 所有指标需与**同时长分桶**的大盘均值对比。15s 视频的完播率和 60s 视频没有可比性。

---

## 5 个内容诊断标签

### 1. 🪝 "标题党"货不对板 (Hook Deception)

**指标组合特征：**
- 3s 完播率极高（超大盘均值 +20% 以上）
- 3s → 5s 遭遇断崖级流失（流失率 > 60%）
- 后续完播率极低

**完播曲线形态：** 前3秒高峰 → 第4秒断崖下跌 → 后续平坦在底部

**观众心理与根因分析：**
开头的"钩子（Hook）"极具视觉冲击力（悬念、猛料、擦边），但进入第 4 秒正片时，画风、语速或内容发生突变，用户感觉被"骗"进来，产生愤怒并瞬间滑走。

本质问题是 **Hook 与正片之间的"承接断裂"**——前3秒建立的情绪预期和内容风格，在第4秒没有得到延续。

**优化师行动指令：**
> **【内容重组】** 锁定并保留前 3 秒神级 Hook 资产。强制要求重拍第 4-8 秒的承接段落，实现"声画一致、情绪不降级"，平滑过渡到卖点。
>
> **具体操作：**
> - 保持前 3 秒不动
> - 第 4-8 秒重新设计过渡段，确保画面风格、语速、BGM 与前 3 秒一致
> - 在过渡段中自然引入产品/卖点，而非突兀切换

---

### 2. 💎 慢热好剧死于片头 (Slow Burn / Hidden Gem)

**指标组合特征：**
- 3s 完播率极低（低于大盘均值 -30%）
- P50 → P100 留存率极高（只要熬过前 3 秒的人，几乎全看完了且买单）

**完播曲线形态：** 前3秒陡峭下滑 → 存活用户几乎不再流失 → 平坦持续到结尾

**观众心理与根因分析：**
视频中后段的痛点挖掘和产品展现极其惊艳，但前 3 秒的引入过于拖沓（比如冗长的企业 Logo、平淡的问候语），导致大量高潜用户根本没耐心看到精华就走了。

这是最让人惋惜的一种素材——**内容是好内容，但被一个烂开头杀死了**。

**优化师行动指令：**
> **【片头前置替换】** 抽取后台高赞/高看节点的第 X 秒高潮画面（倒序剪辑），或直接提取核心痛点加上爆点音效，覆盖替换掉现有的庸常片头。
>
> **具体操作：**
> - 找到视频中用户留存率最高的片段（通常是核心卖点展示段）
> - 将该片段的精华帧/关键画面剪到最前面 3 秒
> - 原有的开头内容（Logo、问候）移到中后段或直接删除
> - 保持中后段内容完全不变（那是这条素材的宝藏）

---

### 3. 😴 叙事催眠 / 视觉疲劳 (Pacing Drag)

**指标组合特征：**
- 3s 完播正常
- 5s → P50 呈现"均匀线性下滑"（没有断崖，但一路狂泄）

**完播曲线形态：** 平滑的斜坡式下降，没有明显的拐点

**观众心理与根因分析：**
视频开局正常，但内容呈现方式极其枯燥。通常是因为"一镜到底、单机位、画外音平缓无起伏、缺乏视觉刺激"，用户看着看着睡着滑走了。

核心问题不是内容不好，而是 **呈现节奏太单调**——人的注意力天然需要"变化"来维持，一成不变的画面会让大脑进入自动忽略模式。

**优化师行动指令：**
> **【高频阻断器 (Pattern Interrupt)】** 保持解说音频不变。要求后期每隔 3-5 秒加入一个视觉刺激点（切机位、放大特写、加入 B-roll 空镜头、砸大字报、加突发音效），人为制造兴奋节奏。
>
> **具体操作：**
> - 解说词/旁白不用改（信息是对的）
> - 每 3-5 秒必须有一次画面变化：切景、缩放、文字弹入、转场特效
> - 在关键卖点处加入视觉强调（放大、高亮、暂停 0.5 秒）
> - 可选：在节奏平淡处加入"叮"一声的音效提示

---

### 4. 🍿 沉浸式爆米花 / 忘加按键 (Entertainment Trap)

**指标组合特征：**
- P50 及整体完播率极高
- **CTR（点击率）极低**

**完播曲线形态：** 全程平坦高位，几乎没有流失

**观众心理与根因分析：**
视频内容极为好看（极度搞笑、感人剧情、完美爽文），用户完全把它当做"原生短视频"看得很爽，却忘记了这是一个广告，缺乏商业转化的"临门一脚"。

**内容质量没有问题，问题是缺少商业转化触发器。** 用户看完了，满意了，然后……就划走了。没有任何元素提醒他们"这是个广告，你可以点击购买"。

**优化师行动指令：**
> **【强宣发贴片】** 内容无需改动（资产极佳）。强制在视频的 P50 处及最后 3 秒加入极强的商业指令：
>
> **具体操作：**
> - 在 P50 时间点加入一个半透明的产品卡片或价格标签（不打断观看体验，但让用户意识到这是广告）
> - 最后 3 秒加入：大号动态箭头指向左下角 + 主播洗脑口播"点击下方链接立即抢"
> - 可选：在视频中间插入一个 1 秒的"闪屏式"产品展示
> - **不要修改内容主体**——这条素材的沉浸感是它最大的资产

---

### 5. 🪓 粗暴拦截 / 精准洗粉 (Hard Filter)

**指标组合特征：**
- 5s / P50 完播率极低（人全跑光了）
- **CTR / CVR 极高（留下来的人几乎全买单了）**

**完播曲线形态：** 前1-2秒断崖式下跌 → 极少数留存用户直到最后

**观众心理与根因分析：**
极度露骨的打法。视频一上来 1 秒钟就抛出极其严苛的价格或门槛（比如"不是保时捷车主别看了"），不相关的人 1 秒划走，但精准目标人群被瞬间激活并快速点击。他们甚至不需要看完视频。

这不是"内容质量差"，而是一种**高效的受众清洗过滤器**。

**优化师行动指令：**
> **【保持原貌并放量】**
>
> ⚠️ **不要优化完播率！** 系统的常见误区是要求提高完播率。这其实是最高效的"受众清洗过滤器"，直接把这套"劝退式话术"复制到其他产品线上。
>
> **具体操作：**
> - 完播率低是**设计目标**，不是 bug
> - 把这条素材的"开场筛选话术"提炼为模板
> - 在其他产品线上复制该模式：1秒内抛出精准筛选条件
> - 关注的核心指标不是完播率，而是 CTR 和 CVR

---

## 标签匹配决策树

```
Step 1: 检查 3s 完播率
  ├─ 极高 (+20% 大盘) → 检查 3s-5s 流失率
  │   ├─ 流失 > 60% → 【Hook Deception】
  │   └─ 流失正常 → 继续 Step 2
  ├─ 极低 (-30% 大盘) → 检查 P50→P100 留存率
  │   ├─ 极高留存 → 【Slow Burn / Hidden Gem】
  │   └─ 正常/低 → 内容全面失败（非 L2 标签，由 L1 判断）
  └─ 正常范围 → 进入 Step 2

Step 2: 检查完播曲线形态
  ├─ 均匀线性下滑（无断崖） → 【Pacing Drag】
  ├─ 全程高位平坦 → 检查 CTR
  │   ├─ CTR 极低 → 【Entertainment Trap】
  │   └─ CTR 正常/高 → 内容+转化俱佳（非问题标签）
  └─ 前 1-2 秒断崖 + 高 CTR/CVR → 【Hard Filter】
```

当无法明确匹配任何 L2 标签时（例如完播曲线形态不典型），标注为"L2 未匹配明确模式"并跳过 L2 标签，不要强行归类。
