# L1 宏观转化诊断：素材生命周期阶段（Lifecycle Stage）

## 诊断视角

从**转化漏斗**的角度审视素材的商业表现：

```
Spend → Impressions → Clicks → Conversions → ROAS
```

通过**静态指标阈值**（当前快照）与**动态时间导数**（趋势加速度）的双重判断，将素材定位到 6 个生命周期阶段之一。

---

## 关键概念：一阶导数与二阶导数

- **一阶导数（斜率）**：指标随时间的变化方向。一阶导 > 0 = 在涨，< 0 = 在跌。
- **二阶导数（加速度）**：指标变化速度本身的变化。二阶导 > 0 = 涨/跌的速度在加快（或减速下跌），< 0 = 涨/跌的速度在减缓（或加速下跌）。

结合静态值与动态导数，才能区分"正常下跌"和"断崖暴雷"。

---

## 6 个生命周期标签

### 1. 🚨 杀猪盘断崖暴雷期 (Sudden Crash)

**静态触发条件：**
- CTR 极高（处于大盘前 10% 分位）
- CVR 极低（处于大盘后 10% 分位）
- ROAS < 0.8（正在亏钱）

**动态导数特征：**
- 一阶导数 < 0（回报直线下降）
- 二阶导数 < 0（跌落速度在加快，呈断崖状）

**现象解读：**
素材靠夸张/低价骗点击洗入大量伪劣流量。算法前期误判给量，随后发现转化极差，遭到媒体降权严惩，成本瞬间暴增，形成断崖式亏损。

**根因分析：**
CTR 与 CVR 的极端反差是"杀猪盘"的核心特征——用户被"骗"进来但完全不买单。算法在快速纠错，降权速度会越来越快。

**投手行动指令：**
> **【绝对止损】** 无脑立即关停。此类素材无自然反弹可能，不要带有"再观察一天"的侥幸心理。每多跑一小时都是在烧钱。

---

### 2. 🚀 指数爆发期 / 飞轮期 (Rocket Launch)

**静态触发条件：**
- Spend 激增（突破大盘均值，进入放量快车道）
- ROAS > 1.2（丰厚盈利）

**动态导数特征：**
- 一阶导数 > 0（回报在涨）
- 二阶导数 > 0（且涨得越来越快，呈指数向上）

**现象解读：**
罕见的超级爆款。精准打中核心受众，且击穿了算法推荐的超级流量池阈值（ECPM 极高），进入"量价齐升"的完美飞轮。

**根因分析：**
素材内容与目标受众高度匹配，算法持续加码曝光。ECPM 极高意味着平台认为你的广告对用户有价值，愿意把最好的流量给你。这是一个正反馈循环。

**投手行动指令：**
> **【火力全开】** 立刻放开该计划的所有预算上限，严禁修改哪怕一个字的定向或出价（防模型破坏）。提炼剪辑框架作为 S 级资产。
>
> **关键提醒：** 这种状态极其珍贵且脆弱。任何对计划设置的修改（调出价、改定向、换素材标题）都可能打断算法的正反馈循环，导致飞轮立刻停转。保护这个计划像保护眼睛一样。

---

### 3. ⚠️ 见顶疲劳期 (Growth Plateau)

**静态触发条件：**
- Spend 处于高位
- ROAS ≈ 1.0（尚能保本）

**动态导数特征：**
- 一阶导数 > 0 或趋近 0（还在涨，但涨到了天花板）
- 二阶导数 < 0（动能严重衰竭，增速在减慢）

**现象解读：**
素材曾经是爆款，但目标人群包已被洗了多次，用户开始产生审美疲劳或免疫。系统探索新人群阻力极大。

**根因分析：**
人群覆盖饱和 + 内容新鲜感衰减。算法还在努力给量（所以一阶导还是正的），但每一轮新增用户的转化效率都在下降（二阶导为负），说明素材的有效触达已经见顶。

**投手行动指令：**
> **【拉响警报】** 不要马上关，但立刻安排剪辑师保留核心逻辑，更换该素材的前 3 秒 Hook、BGM 或配图，生成微调变体（变种素材）上架接盘。
>
> **监控指标：** 紧盯 ROAS 是否跌破 1.0。一旦跌破，从"警报"升级为"软着陆"阶段处理。

---

### 4. 🪂 平缓衰退期 / 软着陆 (Soft Landing)

**静态触发条件：**
- Spend 逐渐变少
- ROAS < 1.0（盈亏线边缘徘徊）

**动态导数特征：**
- 一阶导数 < 0（回报在降）
- 二阶导数 > 0（虽然在跌，但跌势放缓，趋于一个平稳的低保点）

**现象解读：**
流量红利基本退潮，进入长尾期。流量池被稳定榨取，虽然不惊艳，但在某个低阶出价下仍有一丝空间。

**根因分析：**
素材已经过了最佳周期，但内容本身不差，只是受众池快榨干了。二阶导为正说明下跌正在收敛，素材会在一个低位稳定下来而不是继续崩盘。

**投手行动指令：**
> **【防守收割】** 降级为长尾防守素材。尝试下调转化出价（CPA Bid），用更廉价的成本榨干它的最后一滴残余价值，跌破红线再关。
>
> **底线设定：** 设置一个明确的 ROAS 底线（如 0.7），跌破即关停，不要无限期"养老"。

---

### 5. 🧟 系统停量期 / 僵尸池 (Algorithm Zombie)

**静态触发条件：**
- Spend 趋近于 0
- 即便提价也基本无曝光

**动态导数特征：**
- 样本量过低，不足以计算导数
- 各项增速长期为 0

**现象解读：**
创意方向彻底失败，或者极其疲劳。其 ECPM 过低，在竞价队列中连展示机会都买不到，彻底被算法抛弃。

**根因分析：**
系统已经给这条素材"判了死刑"。提价没用——问题不是出价不够高，而是算法预估你的转化率极低，再高的出价乘以极低的预估转化率，ECPM 依然不够看。

**投手行动指令：**
> **【断舍离】** 直接归档或删除。留在账户里不仅占坑位，还会拉低账户的整体预期模型评分。
>
> **注意：** 有些投手会尝试"复活"僵尸素材（改标题、换封面）。不要这样做——算法对素材 ID 的历史表现有记忆，改表面元素无法重置系统评分。如果内容本身有价值，用新账户或新计划重新上传。

---

### 6. 🌸 流量诈尸期 / 二春翻红 (Traffic Renaissance)

**静态触发条件：**
- 历史消耗长期低迷
- 突然 CVR 异常爆发

**动态导数特征：**
- 一阶导数突然 > 0 且成倍数跳跃式激增

**现象解读：**
"老树发新芽"。可能是因为近期短视频热点变迁、季节气候改变引发突然的需求，被系统算法重新"捞"了出来。

**根因分析：**
外部环境变化（热点话题、季节需求、竞品退出）导致原本沉寂的素材重新匹配到了一波高意向人群。这是算法的"二次发现"机制在起作用——系统会周期性地给老素材少量探索流量，如果转化率突然飙升，就会加大分配。

**投手行动指令：**
> **【抢抓机遇】** 马上注入隐藏预算。必要时甚至可以原封不动把该视频新建一个复制计划，趁着神秘的算法春天狠捞一笔。
>
> **关键动作：** 不要修改原素材的任何设置。新建复制计划 + 加预算，让原计划和复制计划同时跑，双倍吃红利。同时密切监控——"二春"窗口期通常很短（3-7天），要快速决策。

---

## 标签匹配优先级

当多个标签的触发条件存在交叉时，按以下优先级判断：

1. **Sudden Crash** — 最高优先级（止损第一）
2. **Algorithm Zombie** — 次高优先级（已死，不需要进一步分析）
3. **Rocket Launch** — 第三优先级（保护飞轮）
4. **Traffic Renaissance** — 第四优先级（抓住窗口期）
5. **Growth Plateau** — 第五优先级
6. **Soft Landing** — 最低优先级（最温和的状态）

如果一条素材同时满足多个标签的部分条件，选择**匹配度最高**（满足条件数量最多）的标签。如果匹配度相同，按上述优先级选择。
